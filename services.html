We build websites using Content Management Systems and hand-coding websites.

Content Management Systems.

    Benefits

  • You don't need to be able to code to use them. This is really useful if you intend to update the website yourself.
  • If you want to be able to update the site regularly with new content, CMS are really easy to use.

    • Disadvantages

  • The downside is they can be 'bloated' with code. Slowing the website down.
  • Search Engine Optimisation can be challenging as you can't get to the code easily and will need to install plugins. CMS and the plugins will need updating regularly.
